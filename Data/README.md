# 📂 Dataset Information

This folder contains the datasets used to build the **CRM Sales Pipeline Dashboard** in Power BI.

## Dataset Source

The dataset was obtained from **Kaggle** for educational and portfolio purposes.

**Dataset Name:** Predictive Analytics for CRM Sales Performance

**Author:** Agung Pambudi

**Kaggle Link:**  
https://www.kaggle.com/datasets/agungpambudi/crm-sales-predictive-analytics

## Files Included

- accounts.csv
- products.csv
- sales_pipeline.csv
- sales_teams.csv
- metadata.csv

## Purpose

These datasets were used for:

- Data Modeling
- Power Query Transformations
- DAX Measure Creation
- KPI Development
- Sales Performance Analysis
- Customer Analytics
- Interactive Dashboard Design
