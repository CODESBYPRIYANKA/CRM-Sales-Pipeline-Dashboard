# 📊 CRM Sales Pipeline Dashboard

## 📌 Project Overview

This project is an interactive CRM Sales Pipeline Dashboard built using Power BI to analyze sales performance, customer outcomes, and pipeline efficiency.

The dashboard provides executives and sales managers with key performance indicators (KPIs), trend analysis, and actionable insights to support data-driven business decisions.
